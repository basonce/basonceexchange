Frequently Asked Questions

How do I deposit USDT to BASONCE?

Log in, go to Assets → Deposit → select USDT → choose your network (TRC20 recommended for lowest fees) → copy the address → send from your external wallet.

Which USDT network should I use for deposits?

TRC20 (Tron network) is recommended for lowest fees and fastest confirmation times. Always verify the network matches before sending.

Can I withdraw crypto from BASONCE?

Yes. Withdraw any cryptocurrency to an external wallet address. Go to Assets → Withdraw → select coin → enter address and amount → confirm.

Is my crypto safe in the BASONCE wallet?

Yes. BASONCE uses cold wallet storage for the majority of funds, two-factor authentication and 24/7 security monitoring to protect all user assets.
