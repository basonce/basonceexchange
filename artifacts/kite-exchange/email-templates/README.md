# BASONCE Email Templates

Ready-to-paste templates for Supabase Auth → Email Templates.

URL: https://supabase.com/dashboard/project/jfjjymprvjfltpvmfptj/auth/templates

For each template below:
1. Click the tab name (e.g. "Confirm signup")
2. Replace the **Subject** with the one given
3. Replace the **Message body (HTML)** with the HTML below
4. Click **Save changes**

All templates use the BASONCE logo from: `https://basonce.com/basonce_logo_son_biten.png`
